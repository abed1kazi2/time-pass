# Abed Kazi — Digital Portfolio A professional student portfolio website showcasing achievements, certifications, projects, and contact information. 🌐 **Live Site:** [https://abedkazi.github.io/portfolio](https://abedkazi.github.io/portfolio) --- ## Features - **Dark / Light mode toggle** - **Scroll progress bar** - **Typed text animation** in hero section - **Scroll-reveal animations** on all sections - **Lightbox gallery** viewer - **Mobile responsive** with hamburger menu - **Sections:** Home · About · Achievements · Projects · Gallery · Contact --- ## Folder Structure ``` portfolio/ ├── index.html ← Main website file ├── README.md ← This file └── assets/ ← All images go here ├── photo.jpg ← Your profile photo ├── cert-matlab.jpg ← MATLAB Coursera certificate ├── cert-physics.jpg ← Physics Coursera certificate ├── cert-netcad.jpg ← NetCAD certificate ├── cert-yuvaal.jpg ← Yuva AI certificate ├── cert-ai4a.jpg ← AI4A certificate ├── project-drone.jpg ← BWB Drone photo ├── project-ai4a.jpg ← AI4A project screenshot ├── design-thinking.jpg ← Design Thinking photo ├── ace-award.jpg ← ACE award photo └── event1.jpg ← Any event photo ``` --- ## How to Deploy on GitHub Pages 1. Create a new GitHub repository named `portfolio` (or `.github.io`) 2. Upload all files: `index.html`, `README.md`, and the `assets/` folder with your images 3. Go to **Settings → Pages** 4. Under **Source**, select `main` branch and `/ (root)` folder 5. Click **Save** — your site will be live in ~2 minutes at `https://.github.io/portfolio` --- ## Personalising the Portfolio Open `index.html` and update: | What | Where in the file | |------|-------------------| | Your email & phone | `#contact` section social links | | LinkedIn URL | `href="https://linkedin.com/in/abedkazi"` | | GitHub URL | `href="https://github.com/abedkazi"` | | Profile photo | Replace `assets/photo.jpg` and update the `hero-avatar` div | | Gallery images | Update `onclick="openLightbox('assets/...')"` paths | | Certificate images | Add `` inside `.cert-card` divs | --- ## Certificates Included | Certificate | Issuer | Date | |-------------|--------|------| | Introduction to Programming with MATLAB | Vanderbilt University / Coursera | Dec 2025 | | Engineering Physics Prerequisites | MIT Vishwaprayag University / Coursera | Sep 2025 | | NetCAD Network Academy | NetCAD | 2025 | | Yuva AI | Government of India | 2025 | | AI for All (AI4A) | National Initiative | 2025 | --- ## Built With - Pure HTML5, CSS3, JavaScript (no frameworks) - Google Fonts: Syne + DM Sans - Fully self-contained single file --- *© 2025 Abed Kazi · Aurangabad, Maharashtra*
